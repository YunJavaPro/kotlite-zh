# Kotlite 中文文档

这是 Kotlite 编程语言及其解释器的中文文档仓库。

## 什么是 Kotlite

*Kotlite* 是一个开源的类型安全编程语言，它拥有 Kotlin 编程语言脚本变体的丰富子集。它附带了标准库，这些库是 Kotlin Multiplatform/Common 标准库和一些第三方库的子集。

*Kotlite 解释器* 是一个轻量级的 Kotlin Multiplatform 库，用于解释和执行用 Kotlite 编写的代码，并在主机运行时环境和嵌入式运行时环境之间建立桥梁。
